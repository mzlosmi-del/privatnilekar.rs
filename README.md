# PrivatniLekar.rs

Platforma za zakazivanje lekarskih pregleda u Srbiji.

---

## 🚀 Kako postaviti sajt (korak po korak)

### 1. GitHub — čuvanje koda

1. Idite na **github.com** → prijavite se ili napravite nalog (besplatno)
2. Kliknite **"+"** (gore desno) → **"New repository"**
3. Ime repozitorijuma: `privatnilekar`
4. Postavite na **Public**
5. Kliknite **"Create repository"**
6. Kliknite **"uploading an existing file"**
7. Prevucite sve fajlove iz ovog foldera (`index.html`, `vercel.json`, `README.md`)
8. Kliknite **"Commit changes"**

---

### 2. Vercel — hosting (besplatno)

1. Idite na **vercel.com**
2. Kliknite **"Sign up"** → odaberite **"Continue with GitHub"**
3. Kliknite **"Add New Project"**
4. Pronađite repozitorijum `privatnilekar` → kliknite **"Import"**
5. Ništa ne menjajte → kliknite **"Deploy"**
6. Za ~30 sekundi sajt je živ na adresi: `privatnilekar.vercel.app`

✅ **Gotovo! Sajt je online.**

---

### 3. Svaki put kad ažurirate sajt

1. Idite na GitHub → vaš repozitorijum `privatnilekar`
2. Kliknite na `index.html`
3. Kliknite na ikonu olovke (Edit)
4. Napravite izmene
5. Kliknite **"Commit changes"**
6. Vercel automatski detektuje izmenu i za 30 sekundi sajt je ažuriran

---

### 4. Vaš domen privatnilekar.rs

Kad kupite domen na **rnids.rs**:

1. U Vercel → vaš projekat → **Settings** → **Domains**
2. Ukucajte `privatnilekar.rs` → kliknite **Add**
3. Vercel prikazuje 2 DNS zapisa (izgledaju otprilike ovako):
   - `A` record → `76.76.21.21`
   - `CNAME` record → `cname.vercel-dns.com`
4. Idite na **rnids.rs** → DNS podešavanja vašeg domena
5. Dodajte ta 2 zapisa
6. Sačekajte 10–30 minuta
7. `privatnilekar.rs` sada otvara vaš sajt ✅
